도망가는 말티즈

도망가는 말티즈를 3초 안에 한 마리라도 잡으면 클리어!

잡은 말티즈 0
남은 시간 3s

🎯 목표: 말티즈 1마리만 잡아도 미션 성공!

말티즈가 번개처럼 나타났다 사라져요!

게임 방법

  1. 게임 시작을 누르면 타이머가 3초에서 출발합니다.
  2. 놀이판 어딘가에 말티즈 얼굴이 순간이동하듯 나타났다 사라집니다.
  3. 말티즈를 클릭하면 잡은 마리 수가 올라가고, 다음 장소로 도망가요.
  4. 3초 안에 단 1마리만 잡아도 임무 성공! 일시정지 버튼으로 숨 돌릴 수 있어요.
준비됐다면 게임 시작을 눌러 도망가는 말티즈를 잡아보세요!

v1은 아이디어·디자인·코드·밸런스 요구사항을 모두 합친 결과물입니다.
