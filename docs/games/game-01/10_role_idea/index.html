게임 방법

  1. 게임 시작 버튼을 누르면 타이머가 30초에서 시작돼요.
  2. 보라색 놀이판 어디엔가 말티즈가 불쑥 나타납니다.
  3. 말티즈를 빠르게 연타하면 더 화가 나고 점수를 얻을 수 있어요.
  4. 점수 1점을 얻으면 말티즈가 더 교활해진 다음 단계로 넘어가요.
  5. 시간이 0초가 되면 최종 점수가 기록됩니다. 단계가 올라갈수록 난이도가 높아져요!
