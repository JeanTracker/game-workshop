도망가는 말티즈

도망가는 말티즈를 한 마리 이상 잡으면 미션 성공이에요!

점수 0
남은 시간 30s

🎯 목표: 말티즈 1마리만 잡아도 통과!

말티즈가 순식간에 나타났다 사라집니다!

게임 방법

  1. 게임 시작 버튼을 누르면 타이머가 30초에서 시작돼요.
  2. 보라색 놀이판 어디엔가 말티즈가 갑자기 나타납니다.
  3. 말티즈를 클릭하면 “잡은 마리 수”가 올라가고 바로 다른 곳으로 도망가요.
  4. 시간이 0초가 되면 최종 잡은 마리 수가 기록되고, 1마리 이상이면 미션 성공!
준비가 되면 게임을 시작해 보세요!

다음 역할 실습(아이디어/디자인/코드/밸런스)은 이 베이스 버전을 복사해서 시작합니다.
