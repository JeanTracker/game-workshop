게임 방법

  1. 게임 시작 버튼을 누르면 타이머가 30초에서 시작돼요.
  2. 보라색 놀이판 안에 별이 무작위로 나타납니다.
  3. 별을 클릭할 때마다 점수가 올라가고, 별은 다시 다른 곳으로 순간이동해요.
  4. 시간이 0초가 되면 최종 점수가 기록돼요.
준비가 되면 게임을 시작해 보세요!
